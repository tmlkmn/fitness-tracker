# FitMusc

Kişisel fitness antrenman ve beslenme takip uygulaması.

## Tech Stack
- Next.js 16 (App Router)
- React 19 + TypeScript
- shadcn/ui + Tailwind CSS v4
- Neon PostgreSQL + Drizzle ORM
- TanStack React Query
- better-auth (invite-only)
- Anthropic Claude (AI features)
- PWA Support (Serwist)
